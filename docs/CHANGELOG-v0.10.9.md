## 1. 변경 사항 ([PipelineProgressPage.tsx](../src/pages/PipelineProgressPage.tsx))

### 1.1 검사 범위 밖(B-3) 배너 문구 정리
- 의미가 엉켜 있던 문구("미검사 항목"인데 실제론 탐지된 건 등)를 간결·직관적으로 재작성.
- 제목: "검사 범위 밖에서 취약점 N건이 추가로 발견되었습니다."
- 설명: "배포 판정은 선택한 M개 항목만 기준입니다. 이 N건은 판정에 포함되지 않았을 뿐, "안전"을 뜻하지는 않습니다."
- 버튼: "미검사 항목 보기" → "추가 발견 항목 보기".

### 1.2 코드 품질 점수 = 보안 점수와 동일 표시
- 백엔드 `code_quality_score`가 부정확(항상 100/값 불일치)해, 코드 품질 점수를 위 보안 점수 게이지 값(`score`)과 동일하게 표시하도록 변경.
- 미사용이 된 `codeQualityScore` 변수 제거. api.ts 파싱은 원복(순변경 없음).
